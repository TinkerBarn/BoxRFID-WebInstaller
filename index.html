BoxRFID Web Installer

Mit diesem Installer kannst du die BoxRFID-Firmware direkt über den Browser auf dein ESP32-Board installieren.

Installation

Unterstütztes Board:
ESP32-2432S028R (CYD) – BoxRFID V2.0

Wichtige Hinweise:

  • Verwende Chrome oder Edge.
  • Verwende ein datenfähiges USB-Kabel.
  • Falls keine Verbindung möglich ist, Board neu anstecken und erneut versuchen.
  • Falls nötig, beim Verbinden die BOOT-Taste gedrückt halten.
  • Mit der Installation wird die Firmware auf dem Gerät überschrieben.
